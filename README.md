# 🌍 Global Search Trends Dashboard

A React-based dashboard showing real-time global search trends from the last 24 hours.

## 🚀 Quick Start

### Option 1: Direct HTML File
Simply open `index.html` in your web browser.

### Option 2: Local Server
```bash
# Install dependencies (if any)
npm install

# Start the server
npm start
```

Then open [http://localhost:3000](http://localhost:3000) in your browser.

## 📊 Features

- **Real-time Trends**: Last 24 hours search data
- **Interactive Filtering**: Filter by category (Sports, Technology, Science, etc.)
- **Search Functionality**: Find specific trends
- **Visual Analytics**: Color-coded categories and growth indicators
- **Responsive Design**: Works on desktop and mobile

## 🔍 Data Sources

- Google Trends API (simulated)
- Last 24 hours trending searches
- Global search patterns
